Mini Kanban Moderno

Objetivo

Desenvolver uma aplicação web simples para gerenciar uma lista de tarefas diárias utilizando o `localStorage` 
do navegador, permitindo que as tarefas sejam salvas de forma persistente e facilmente organizadas em colunas.



Funcionalidades

* Cadastro de Tarefas: Adicione novas tarefas diretamente no Kanban, escolhendo a coluna inicial (`Fazer`, `Fazendo` ou `Feito`).
* Armazenamento Permanente: Todas as tarefas são salvas no `localStorage` do navegador, garantindo que permaneçam mesmo após fechar ou atualizar a página.
* Remoção de Tarefas: Cada tarefa possui um botão de exclusão que permite removê-la individualmente.
* Limpeza Completa: Um botão permite apagar todas as tarefas armazenadas de uma vez só.
* Marcar como Feito: Tarefas em andamento podem ser movidas rapidamente para a coluna `Feito` com um botão dedicado.
* Filtro de Tarefas: Pesquise tarefas digitando palavras-chave para localizar rapidamente itens específicos.
* Interface Moderna e Responsiva: Layout organizado em colunas, cores indicativas por status, animações suaves ao adicionar ou remover tarefas, e tema escuro alternável.
* Contadores por Coluna: Mostra o número de tarefas em cada coluna para fácil visualização do progresso.



Tecnologias Utilizadas

* HTML5
* CSS3 (com Bootstrap 5)
* JavaScript (ES6+)
* `localStorage` para persistência de dados
